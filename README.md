# iTircher

A website written in noir. 

## Install

```bash
lein deps
bundle install
```


## Usage

Generate javascripts from clojurescripts

'''bash
lein cljsbuild once # Builds the JavaScript files once
lein cljsbuild auto # Watches for changes and automatically builds the
JavaScript files
```

Start webserver

'''bash
lein run
```
